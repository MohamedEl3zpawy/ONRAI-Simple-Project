# 🎮 ON-RAI (Thunder Demon) - E-Commerce Website

A sleek, modern, and minimalist e-commerce web platform designed for premium gaming gear, PC setups, and tech accessories. Built with a focus on eye-comfort and clean digital design.

## ✨ Features
* **Minimalist UI/UX:** Clean aesthetics with a "less is more" approach for a comfortable visual experience.
* **Multi-Page Layout:** Fully structured multi-page navigation for a seamless shopping journey.
* **Responsive Design:** Optimized for modern PC setups and desktop viewing.

## 🛠️ Technologies Used
* **HTML5:** Semantic structure for high web accessibility.
* **CSS3:** Custom styling, layouts, and smooth typography.
* **Notepad++:** Core development environment.
